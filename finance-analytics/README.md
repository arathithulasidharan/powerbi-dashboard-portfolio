# 💰 Finance Analytics Dashboards

This section showcases interactive Power BI dashboards developed to monitor financial performance, invoice tracking, cash flow, accounts receivable, and invoice planning. These dashboards provide stakeholders with actionable financial insights to support informed business decisions.

---

# 1️⃣ Invoice Analytics Dashboard

### 📌 Overview

Designed to monitor invoice performance by providing insights into invoice generation, billing status, customer-wise invoices, and overall financial activity.

### 🛠️ Tools Used

- Power BI
- DAX
- Power Query
- Microsoft Excel

### 📊 Key KPIs

- Total Invoices
- Invoice Value
- Customer Distribution
- Invoice Status
