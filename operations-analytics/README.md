# 🏭 Operations Analytics Dashboards

This section showcases Power BI dashboards developed to monitor project execution, site-level activities, and operational performance. These dashboards help stakeholders track project progress, monitor site status, and gain real-time visibility into ongoing operations.

---

# 1️⃣ Project Site Tracker Dashboard

### 📌 Overview

Designed to monitor project execution across multiple sites by providing a centralized view of project status, site-wise progress, and operational performance. The dashboard enables stakeholders to monitor ongoing projects, identify delays, and make informed decisions for efficient project management.

### 🛠️ Tools Used

- Power BI
- DAX
- Power Query
- Microsoft Excel

### 📊 Key KPIs

- Active Projects
- Active Sites
- Project Status
- Site-wise Progress
- Completion Percentage

### ✨ Key Features

- Interactive KPI Cards
- Dynamic Filters & Slicers
- Site-wise Performance Analysis
- Project Progress Monitoring
- Drill-through Analysis
